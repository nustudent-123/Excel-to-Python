# 📊 Amazon Sales Data Analysis using Python

A beginner-friendly data analysis project developed using **Python, Pandas, and Matplotlib** to analyze Amazon sales data and generate business insights through KPIs, visualizations, and reports.

---

## 📌 Project Overview

This project analyzes an Amazon sales dataset to answer important business questions, identify sales trends, evaluate customer purchasing behavior, and support data-driven decision-making.

The analysis includes:

- Data Cleaning
- KPI Calculation
- Sales Analysis
- Customer Analysis
- Product Analysis
- Payment Method Analysis
- Monthly Sales Trend
- Business Recommendations
- Data Visualization

---

## 🚀 Features

- Load and clean Amazon sales data
- Calculate key performance indicators (KPIs)
- Analyze sales by state and category
- Identify Top 10 Products
- Identify Top 5 Customers
- Analyze customer payment methods
- Monthly sales trend analysis
- Generate business insights
- Create charts automatically

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/Amazon-Sales-Analysis.git
```

2. Move into the project directory

```bash
cd Amazon-Sales-Analysis
```

3. Install dependencies

```bash
pip install pandas matplotlib
```

4. Run the analysis

```bash
python analysis.py
```
